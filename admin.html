How to publish changes
Update your live dashboard in 3 steps
1
Click Copy JSON or Download file above to get your updated nys-grants.json
2
Go to your GitHub repo — github.com/amberlucky76/grant-3 — and click on nys-grants.json
3
Click the pencil icon to edit, select all the existing text, paste in your new JSON, then click Commit changes. Your dashboard updates in ~30 seconds.
Tip
Bookmark this admin.html page so you can always come back to add or remove grants without touching code. Load your existing JSON first so you don't lose previous entries.
